Características da CSS

Unidades de medida

Absolutas:

  • pixels (px)
  • points (pt)
  • inch (in)
  • centímetros (cm)
  • milímetro (mm)

Relativas: (efeito casccata)

  • em (sempre baseia no tamanho do font-size anterior)
  • rem (se refere ao primeiro elemento "root" ou seja a tag html)
  • porcentagem (0%)
  • ex (tamanho conforme a letra "x" da fonte)
  • ch (obs: só funciona em fontes monoespaçadas...que tem o mesmo espaço , baseia no tamanho da fonte)

Obs: O tamanho padrão do font-size do html é de 16px.

Viewport

Espaço disponível visível e usável para o usuário.
vh = viewport height
vw = viewport width

Inherit

Forçar uma herança de algum item como uma fonte ou um tamanho.

Cores

Podem serem aplicadas pelos nomes (white, black, gray), em exadecimal indo de 0 à 9 e de A à F(#FFF,#000,#CCC), pelos aspectos RGB que vão de 0 à 255 (rgb(255,255,255),rgb(0,0,0),rgb)(80,80,80)), HSL porcentagem das tonalidades das cores de 0% à 1000% (hsl(100%,100%,100%)) e também existe o currentColor que herda a cor do item pai.

Imagens Responsivas

Propriedade "Object-fit"

Imagem original (vou deixar comentada)
Imagem com o "Object-fit: fill;"

Obs: A imagem pode se destorcer para que caiba em determinado espaço.

Imagem com o "Object-fit: contain;" ou "scale-down"

Obs: Diminui a imagem sem destocer-la até que caiba no espaço definido(o scale-down diminui tbm a qualidade).

Imagem com o "Object-fit: cover;"

Obs: Ajusta o tamanhho e corta a imagem para manter a proporção, geralmente o corte centralizado.

Imagem com o "Object-fit: none;"

Obs: Não muda o tamanho da imagem somente corta na proporção definida.

CSS @media

O @media é acionado quando se satisfaz alguma condição imposta, e é assim que fazem o site ser mais responsivo...(tente mexer na proporção)

Mudar para impressões

@media only print {
     Modificações...
}

Obs: Essas modificações somente ocorrerão quando for imprimir a página.

Orientation

O @media não identifica a orientação do dispositivo, ele trabalha com as proporções da tela.
